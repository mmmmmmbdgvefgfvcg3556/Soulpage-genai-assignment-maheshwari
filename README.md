# Conversational Knowledge Bot

This project implements a conversational AI chatbot using LangChain.
The bot can interact with users, use tools to fetch information,
and maintain conversation memory.

## Features
- Conversational chatbot
- Web search using Tavily tool
- Conversation memory
- Built with LangChain and OpenAI

## Tools and Memory
- Tavily Search Tool for knowledge retrieval
- ConversationBufferMemory for context retention

## How to Run
```bash
pip install -r requirements.txt
python app.py
